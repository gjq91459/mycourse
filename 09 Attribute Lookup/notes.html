Attribute Lookup

Key Points
  • Lookup different for read and write
  • Read lookup traverses the dictionary hierarchy
  • Write lookup on looks in object dictionary
  • Can override normal lookup procedure - descriptor protocol, special methods
Descriptor protocol
  • __get__()
    • called on attribute read
  • __set__()
    • called on attribute write
Special Methods
  • __getattr__()
    • called if attribute not found
  • __getattribute__()
    • - always called if attribute is read
  • __setattr__()
    • always called if attribute is written
