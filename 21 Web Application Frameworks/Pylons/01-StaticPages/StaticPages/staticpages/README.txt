Serve up static pages from the public directory:
	e.g. http://localhost:5000/page1.html
	e.g. http://localhost:5000/page2.html
	e.g. http://localhost:5000/page3.html
	
or mako templates from the template directory:
	e.g.http://localhost:5000/serve/view1
	e.g.http://localhost:5000/serve/view2
	e.g.http://localhost:5000/serve/view3

Change server and port in development.ini
