Modules and Packages

Modules
  • Single file - import to use
  • Must be on Python Path
Packages
  • Group of files - import to use
  • Must be on Python Path
  • Can be contained in sub-directories
  • Each directory must contain the marker file:
    • __init__.py
Key Points
  • import package/module
    • names kept in separate namespace
  • from package/module import names
    • names imported into local namespace
  • namespaces - symbol tables stored as dictionaries
  • don't use wildcard import
