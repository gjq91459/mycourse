Introduction

What is Python?
  • Scripting language - nowadays used for large projects
  • Dynamic - everything evaluated at runtime
  • Interpreted - but code is compiled to byte code first
  • Object Oriented and Functional - hybrid language
  • Uses a Virtual Machine - just like Java and C#
What Version?
  • 2.7 - easily the most common version used in industry
  • 3.4 - slow to catch on because not backward compatible
  • Jython - compiles to Java byte code and runs on JVM
  • Iron Python - compiles to .NET byte code and runs in .NET
Key Points
  • Libraries - huge set of libraries
  • Mixing languages - easy to interface to C and C++ code
  • Pure Python is slow - up to 100 times slower than C
  • Performance is good - most code uses libraries written in C or C++
  • Python and Jython - same source code, but different libraries
  • Python Interpreter is single threaded - GIL
